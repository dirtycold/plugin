Class IPlugin

Plugin::IPlugin — Interface of a plugin.

Synopsis

// In header: <Plugin/IPlugin.h>


class IPlugin : private boost::noncopyable {
public:
  // construct/copy/destruct
  ~IPlugin();

  // public member functions
  const std::string & iGetPluginName() const;
  const Vers::Version & iGetPluginVersion() const;
};

Description

You are supposed to inherit and implement this interface in your plugin facade. Your facade factory should be created using the macro PLUGIN_FACTORY_DECLARATION(T) and PLUGIN_FACTORY_DEFINITION(T).

IPlugin public construct/copy/destruct

  1. ~IPlugin();
    Destructor.

    This destructor is declared protected in order to strictly forbid user to delete instance of a plugin facade. Construction and destruction of a plugin facade must all resides inside the boundary of the concrete plugin library.

IPlugin public member functions

  1. const std::string & iGetPluginName() const;
    Get plugin name. 
  2. const Vers::Version & iGetPluginVersion() const;
    Get plugin version.
