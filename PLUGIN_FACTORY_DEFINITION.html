Macro PLUGIN_FACTORY_DEFINITION

PLUGIN_FACTORY_DEFINITION — Defines fonctions to create and destroy your plugin facade.

Synopsis

// In header: <Plugin/PluginFactory.h>

PLUGIN_FACTORY_DEFINITION(T)

Description

Must be used in a cpp file. Your plugin facade will behave as a singleton.

Parameters:

T

It is the concrete type of your plugin facade. T is not required to be in the global namespace.
